Explanation: To create a good tool function, the doc string (comments) should be rich and detailed. The LLM will refer to these comments as we are no longer using external tools.

Key Points:
- Use rich comments in the doc string of the function.
- The LLM will refer to the doc string for understanding the function.
- Make sure the comments are detailed and comprehensive.
