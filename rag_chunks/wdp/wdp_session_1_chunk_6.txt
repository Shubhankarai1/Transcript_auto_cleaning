Explanation:
The email classification system takes an email as input and classifies it as spam or legitimate using AI. The system consists of several functions including `classify_email`, `handle_spam`, `drafting_response`, and `route_email`.

Key Points:
- Email state serves as the input for all functions
- Each function uses the email state to perform its task
- Functions may use different parts of the email state
- The system determines if an email is spam or legitimate by checking the value of the 'is_spam' key in the email state
- The `route_email` function makes decisions based on the value of 'is_spam' and returns either 'spam' or 'legitimate'
