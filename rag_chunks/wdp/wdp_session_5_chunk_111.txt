Explanation: To test the Writer node in Dialogflow, create a state and put a user message inside it. The message should contain customer issues like "I have not received the product in 15 days now. After updating the state manually, call the writer node for drafting a reply based on the updated state values.

Key Points:

- Pass the state to the intake node first
- Update the state manually
- Call the writer node to draft a reply
